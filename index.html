Real-Time Jello

In a team of 2, we implemented a real-time jello simulation with OpenGL and C++. I was responsible for implementing the physics and interactions. The jello is based on a mass-spring system using Euler integration and Runge-Kutta 4th order.

Future improvements:
We would like to implement allow the jello to jiggle of the plate and collide with other objects in the bounding box.

Path Tracer

In a team of 2, we implemented a Monte Carlo Path Tracer that runs purely on the CPU with C++. Our Path Tracer is able to load and render .obj models and support diffuse, specular, gloss and glass materials using bidirectional reflectance distribution functions (BRDF). We accelerated our program using bounding volume hierarchies and OpenMP. We later rendered scenes with different triangle counts and materials to benchmark the results which can be viewed on our Github page.

Future improvements:
We would like to implement a built in Denoiser to reduce the noise in our output, importance sampling to speed up the calculation and possibly a CUDA implementation.

Checkout the Github project here!

Non-PhotoRealistic SPH

In a team of 2 we made a fun program that showcases non-photorealistic rendering with smoothed-particle hydrodynamics (SPH). My partner was responsible for implementing the physics and I was responsible for the visual aspects. The user can select the colors of the outline, background and tones in the non-photorealistic shader. The user can also change the particle sizes and the model displayed.
